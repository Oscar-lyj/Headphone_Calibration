Welcome to the Headphone Calibration Tool

This tool allows you to upload headphone frequency response data (CSV or PNG) and calibrate it against reference curves. You can either upload a CSV file or an image of the frequency curve. Follow the steps below to get started:

  • Upload a CSV file or PNG image of your headphone's frequency response curve.

  • For CSV files:
    • Choose a reference curve and calibration settings.

  • For PNG files:
    • Select a curve extraction mode and sensitivity for extraction.
    • Fill in the minimum and maximum values of the curve.
    • Once upload is pressed, select the region or the point on the pop up window and press submitSelection.

  • For Uploading personal reference curve:
    • Upload a CSV or PNG file as reference.
    • For PNG file, select the mode of extraction and select the region or the point on the preview uploaded image.

  • The results will be shown as a plot and a table/
